## About this Project

This is a Auth System implemented with React and Laravel.

- [React](https://reactjs.org/).
- [Laravel](https://laravel.com/).
- [Laravel Sanctum](https://laravel.com/docs/8.x/sanctum).
- [Bootstrap](https://getbootstrap.com/).
